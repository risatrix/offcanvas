What This Plugin Does

Targets a menu, builds an offcanvas version that hides behind the rest of your page, and adds classes when you hit a designated toggle. Make the browser window small to see the toggle.

NB: You have to add the toggle yourself. The plugin will not do it for you.

What This Plugin Does NOT Do

This plugin does NOT animate via jQuery. It only adds classes that the CSS responds to. Read David Bushell's excellent article for an explanation why.

Use Cases

  • -You have a pre-existing menu and want to make it offcanvas for mobile.(default)
  • -You've already built an offcanvas menu for all size screens and you want to use the toggle functions only.
  • -You want to do something crazy with shifting content around. Good news, everyone: once you've grasped the toggling of classes you can do whatever you want with the CSS!
